About the website

This website is a demonstration of SPA (Single Page Application) Prototype merely using:


  • HTMX
  • Flask
  • Pure Python 3.8

None of the Javascript code is used to render any functionality for dynamic pages

Conclusion


HTMX is elegant if you have simpler application with not much of the dynamica functionalities. Although, it can be used for large-grade applications in conjunstion with Javascript to reduce the codebase by 60% at the same time achieving Server Side Rendering for optimized application.
